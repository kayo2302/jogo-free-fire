# Mini Royale (Protótipo)
Projeto protótipo de um jogo 2D inspirado em battle royale, criado para estudo e hospedagem no GitHub Pages.

## Como rodar
1. Abra `index.html` no navegador (ou use GitHub Pages).
2. Controles:
   - WASD / setas: mover
   - Mouse: mirar
   - Clique esquerdo: atirar
## Observações
- Protótipo educativo: não contém assets de jogos comerciais.
- Para publicar: ativar GitHub Pages na branch `main` (pasta root).
